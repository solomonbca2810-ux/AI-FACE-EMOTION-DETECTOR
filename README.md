**🧠 AI Face Emotion Detection System**

This project detects human emotions in real-time using a webcam and displays the detected emotion on the screen using computer vision and deep learning techniques.

🚀 Features
🎥 Real-time webcam-based emotion detection
😀 Detects multiple emotions (Happy, Sad, Angry, etc.)
⚡ Fast and efficient processing (CPU supported)
🖥️ Live display with emotion labels
🛠️ Tech Stack
Python
OpenCV
Facial Emotion Recognition Library
Computer Vision
📦 Installation

**Install required dependencies:**
pip install opencv-python facial-emotion-recognition

**▶️ How to Run**
python main.py

**📸 Output**
Captures live video from webcam
Detects face and predicts emotion
Displays emotion label on screen in real-time

**⚠️ Requirements**
Working webcam
Python 3.10 recommended
Good lighting for better accuracy
